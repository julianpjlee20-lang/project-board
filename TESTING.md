# 🧪 測試指南

## Cypress E2E 測試

### 什麼是 Cypress？
Cypress 是一個**真實瀏覽器測試工具**,會：
- ✅ 真的開啟瀏覽器
- ✅ 真的點擊按鈕、輸入文字、拖拉元素
- ✅ 確保你的網站在使用者使用時不會出錯

---

## 🚀 如何執行測試

### 方法 1: 視覺化模式（推薦新手）

\`\`\`bash
# 1. 先啟動開發伺服器
pnpm run dev

# 2. 在另一個終端機開啟 Cypress
pnpm run test:open
\`\`\`

**你會看到：**
- 🎬 一個視覺化界面
- 📝 所有的測試列表
- 👁️ 可以點擊測試，看它執行
- ⏯️ 可以暫停、回放每個步驟

---

### 方法 2: 命令列模式（快速執行）

\`\`\`bash
# 1. 先啟動開發伺服器
pnpm run dev

# 2. 在另一個終端機執行測試
pnpm test
\`\`\`

---

## 🔧 常用命令

| 命令 | 說明 |
|------|------|
| \`pnpm run test\` | 執行所有測試（命令列） |
| \`pnpm run test:open\` | 開啟視覺化測試界面 |
| \`pnpm run test:headed\` | 看到瀏覽器執行過程 |
| \`pnpm run test:chrome\` | 用 Chrome 執行測試 |
